# Akhiok (Game Client)

**Akhiok** is a gaming platform, where users can upload games using the Lua scripting language.

## How to use

This project should be used in tandem with *Akhiok Gane Server* (TODO).
This serves as the client side, where the client recievies updates from the server and sends data back (Some data won't be allow to pass to avoid exploits and hacking).

To compile the project use:

```sh
cargo build
```

For release:

```sh
cargo build --release
```

## Features

1. Node Heirachy
2. Lua Scripting
